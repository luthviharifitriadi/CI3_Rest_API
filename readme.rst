HTTP
**************************
* HTTP => Hyepertext Transfer Protocol
* URL => Uniform Resource Locator
  * Alamat dari sebuah resource yang dapat diakses di internet
* GET Meminta sesuatu
* POST mengirimkan request beserta data didalam form
